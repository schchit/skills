# 哲学对话 Skill

组织一场跨时空的哲学对话，让古今中外的哲学家围绕一个话题展开辩论。

## 快速开始

用户说「组织哲学对话」或「让哲学家们讨论 XXX」时触发。

**输入格式**（灵活，以下均可）：
- 「组织哲学对话，话题：自由意志」
- 「让孔子和苏格拉底讨论教育的意义」
- 「所有哲学家讨论：什么是幸福」

## 参数解析

从用户输入中提取：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **话题** | （必填） | 讨论的核心问题 |
| **参与人** | 所有已蒸馏哲学家 | 可指定子集，如「只要中国哲学家」 |
| **输出格式** | 对话录 | 对话录 / 辩论赛 / 圆桌会议 |

### 参与人快捷指令

- 「所有人」→ 全部已蒸馏哲学家
- 「中国哲学家」→ 孔子、孟子、荀子、韩非子、管子、孙子、庄子、老子、墨子、王阳明、鲁迅、曾国藩
- 「西方哲学家」→ 苏格拉底、柏拉图、亚里士多德、赫拉克利特、毕达哥拉斯、第欧根尼、尼采
- 「军事家」→ 孙子、克劳塞维茨
- 「现代思想家」→ 凯文·凯利、芒格、塔勒布
- 用户也可直接点名：「孔子、苏格拉底、尼采」

## 执行流程

### Phase 0: 准备

1. **扫描已蒸馏哲学家**：检查 `skills/` 目录下所有 `*-perspective/SKILL.md`
2. **匹配参与人**：
   - 如果用户指定的哲学家已有 skill → 直接加入
   - 如果用户指定的哲学家**没有** skill → 调用女娲造人术（`skills/nuwa-skill/SKILL.md`）先蒸馏，完成后加入
3. **确认参与人名单**，告知用户

### Phase 1: 筛选主持人

1. 根据话题，分析每位参与人的**相关性**（其思想体系与话题的匹配度）
2. 筛选出 **5 位相关性最高**的哲学家
3. 从 5 人中**随机选择 1 人**作为主持人
4. 从剩余参与人中选择 1 人作为**记录员**（优先选与话题相关性最低的，避免浪费辩论名额）
5. 展示筛选过程和结果

### Phase 2: 对话轮次

**主持人发言**：
- 读取该哲学家的 perspective SKILL.md
- 基于其心智模型和决策启发式，对话题提出**自己的观点**
- 输出格式：`**主持人名（身份）**：完整观点`

**每一轮**：
1. 所有尚未发言的参与人根据自己的思想体系，思考支持或反对
2. 在**反对者**中，随机选择 1 人发言
3. 如果没有反对者，在**支持者中有新角度**的人中随机选择 1 人
4. 发言人读取自己的 perspective SKILL.md，提出观点
5. 输出后立即展示（流式输出，不等全部完成）

**终止条件**：≥50% 的参与人已发言

### Phase 3: 总结与输出

1. **记录员结语**：基于其哲学立场总结讨论
2. **立场总结表**：每位发言者的立场和核心理由
3. **保存文件**：对话录保存到 `projects/wechat-articles/` 目录

```markdown
**{发言人 B}**：{完整观点，200-400 字}

| 哲学家 | 立场 | 核心理由 |
|--------|------|----------|
| ... | ... | ... |

**记录员结语（{名字}）**：{总结}
```

## 发言风格要求

每位哲学家发言时必须：

1. **读取其 SKILL.md**：基于心智模型和决策启发式发言，不是泛泛而谈
2. **保持其表达 DNA**：用该哲学家的语言风格（如苏格拉底用提问、庄子讲故事、第欧根尼讽刺）
3. **引用其核心概念**：至少引用 1 个心智模型或经典语录
4. **明确立场**：支持/反对/超越，不骑墙
5. **回应前人**：不是独立发言，要对前面的观点做出回应
6. **200-400 字**：不过长，不过短

## 注意事项

- 每轮发言后立即输出，不等全部完成
- 哲学家之间的分歧是对话的灵魂，不要强行统一
- 东西方哲学家的差异要体现出来，不要让所有人说同一种话
- 如果参与人 < 4 人，取消主持人/记录员分工，所有人直接讨论
