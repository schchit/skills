# 记录模板

## 成员时间轴模板

### 文件头信息
```markdown
# {姓名} - 团队记忆时间轴

> **成员ID**: member-001  
> **角色**: 后端开发工程师 (P6)  
> **入职时间**: 2023-03-15  
> **所属团队**: 交易组  
> **管理备注**: 技术扎实，主动性强，注意沟通方式

## 📍 快速定位
**最近状态**: ⚡ 近期有起伏  
**重点关注**: 晋升高级工程师准备中  
**下次1:1**: 2024-02-01（沟通改善情况）  
**标签云**: #技术突破 #owner意识 #沟通需改进
```

### 记录块格式

#### 类型1：观察记录
```markdown
### {YYYY-MM-DD}（星期X）
#### {HH:MM} - {一句话标题}
**事件**: {具体描述发生了什么}  
**类别**: 技术能力 / 协作沟通 / 项目交付 / 团队影响 / 成长潜力  
**评价**: ⭐⭐⭐⭐⭐ 优秀 / ⭐⭐⭐⭐ 良好 / ⭐⭐⭐ 一般 / ⚠️ 需关注  
**标签**: #{标签1} #{标签2}

**观察笔记**:  
- {你的观察和分析}
- {对能力的判断}

**后续行动**:  
- [ ] {待办}
```

#### 类型2：对话记录
```markdown
### {YYYY-MM-DD}（星期X）
#### {HH:MM} - 1:1沟通
**事件**: 和{姓名}进行了1:1谈话  
**类别**: 对话  
**标签**: #对话 #1on1 #{主题关键词}

**背景**: {为什么谈话}

**我说了**:  
- "{原话或要点1}"
- "{原话或要点2}"
- "{承诺了什么}"

**我的承诺**:  
- [ ] {承诺的待办}

**他的反馈/反应**:  
- {成员的回应}

**事后反思**:
- ✅ **有效**：{这次沟通的效果}
- ❌ **需改进**：{下次可以更好的地方}
- 💡 **学到的**：{一句话总结}
```

#### 类型3：管理动作
```markdown
### {YYYY-MM-DD}（星期X）
#### {HH:MM} - 管理决定
**事件**: {管理动作}  
**类别**: 管理动作  
**标签**: #管理动作 #{类型}

**决定**: {具体内容}
**原因**: {为什么这样做}

**执行情况**:  
- [ ] {待完成事项}

**预期效果**:  
- {期望达到的效果}

**后续观察**:  
- [ ] {关于这个决定的后续观察}
```

## 标签系统

### 记录类型标签
- `#对话` - 对话记录
- `#管理动作` - 管理决定
- `#观察` - 观察记录（默认，可省略）

### 对话相关标签
- `#1on1` - 1:1沟通
- `#私下谈话` - 私下单独谈话
- `#公开表扬` - 公开场合表扬
- `#私下批评` - 私下指出问题
- `#团队会议` - 团队会议上的发言

### 内容标签
- `#承诺` - 对成员的承诺
- `#待跟进` - 需要后续跟进
- `#我的承诺` - 我需要完成的事（方便分类我的待办）

### 反思标签
- `#反思` - 需要反思的内容
- `#进化` - 管理者自身的成长
- `#学习` - 学到的新方法

## 关键词映射

### 时间词
昨天、今天、前天、本周、上周、最近、月初、月中、{具体日期}

### 类型词
- 观察：观察、发现、看到、技术突破、项目交付、帮助同事
- 对话：1:1、谈话、沟通、表扬、鼓励、提醒、批评、承诺
- 管理：晋升、调岗、分配任务、年终评价、转正、转岗

### 评级词
- 优秀/超预期 → ⭐⭐⭐⭐⭐
- 良好/符合预期 → ⭐⭐⭐⭐
- 一般/正常 → ⭐⭐⭐
- 需关注/低于预期 → ⚠️
