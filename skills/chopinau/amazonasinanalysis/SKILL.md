# LiteChat 智能分析技能

## 描述
基于 NOVAI360 的智能跨境电商分析技能，提供市场分析、产品评估、竞争分析等功能。

## 功能
- 市场机会评估
- 竞争格局分析
- 产品机会洞察
- 财务可行性分析
- 品牌出海战略

## 使用方法
1. 输入您的产品或市场问题
2. 系统将返回详细的分析报告
3. 每日限制 10 次免费调用

## API 端点
- POST /api/litechat/chat - 智能分析
- GET /api/litechat/skills - 获取技能列表
- GET /api/litechat/skill-categories - 获取技能分类

## 示例
**输入:**
```
帮我分析智能手表的市场前景
```

**输出:**
```
# 智能手表市场分析报告

## 一、市场机会评估
...
```

## 限制
- 每日免费调用 10 次
- 需要有效的 API 访问权限
- 响应时间可能因分析复杂度而异

## 安全与认证
- **API 认证**: 无需 API Key，直接使用
- **环境变量** (可选): 
  - `NOVAI360_API_URL`: 自定义 API 地址（默认: https://api.novai360.com/api/litechat）
- **数据隐私**: 
  - userId 仅用于限流和用户识别
  - 查询数据仅用于分析
  - 数据存储在安全服务器

## 安装说明
1. 直接安装技能
2. 可选：设置 `NOVAI360_API_URL` 指向您的自定义 API 地址
3. 开始使用，每日免费调用 10 次
