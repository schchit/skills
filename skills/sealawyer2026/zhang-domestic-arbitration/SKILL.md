# 国内仲裁全流程AI助手 ⚖️

> 专业仲裁律师AI助手 | 仲裁申请 | 答辩 | 庭审 | 裁决执行

## 🎯 适用场景

- 商事仲裁（CIETAC/BAC/SHAC等）
- 劳动仲裁
- 农村土地承包仲裁
- 仲裁条款起草审查
- 仲裁策略制定
- 裁决执行申请
- 裁决撤销/不予执行

## 📋 阶段一：仲裁可行性评估

### 1.1 仲裁vs诉讼选择
```
请分析以下争议，建议仲裁或诉讼：

争议类型：[合同纠纷/股权纠纷/建设工程/等]
争议金额：[金额]
对方当事人：[情况]
合同条款：[是否有仲裁条款]

分析维度：
1. 仲裁条款有效性
2. 仲裁机构选择
3. 仲裁vs诉讼优劣对比
4. 时间成本对比
5. 费用成本对比
6. 保密性需求
7. 执行便利性
8. 最优路径建议
```

### 1.2 仲裁条款审查
```
请审查以下仲裁条款：

[粘贴仲裁条款]

审查要点：
1. 仲裁意愿表示
2. 仲裁事项范围
3. 仲裁机构确定性
4. 仲裁地点约定
5. 仲裁语言约定
6. 仲裁规则选择
7. 条款有效性判断
8. 完善建议
```

### 1.3 仲裁机构选择
```
请推荐适合的仲裁机构：

争议类型：[类型]
争议金额：[金额]
当事人所在地：[地点]
合同履行地：[地点]

推荐分析：
| 机构 | 优势 | 劣势 | 适用性 |
|-----|------|------|--------|
| CIETAC | ... | ... | ... |
| BAC | ... | ... | ... |
| SHAC | ... | ... | ... |
| 其他 | ... | ... | ... |

推荐结论：
```

## 📋 阶段二：仲裁申请

### 2.1 仲裁申请书
```
请起草仲裁申请书：

申请人：[姓名/单位]
被申请人：[姓名/单位]
仲裁机构：[机构名称]
仲裁请求：[具体请求]

申请书应包含：
1. 当事人信息
2. 仲裁依据（仲裁协议）
3. 仲裁请求（明确具体）
4. 事实与理由
5. 证据清单
6. 财产保全申请（如需）
7. 证据保全申请（如需）
```

### 2.2 仲裁答辩书
```
请起草仲裁答辩书：

被申请人：[姓名]
申请人：[姓名]
仲裁请求：[请求内容]

答辩书应包含：
1. 被申请人信息
2. 对仲裁请求的回应
3. 事实与理由
4. 反请求（如有）
5. 证据支持
6. 答辩结论
```

### 2.3 仲裁反请求
```
请起草仲裁反请求申请书：

反请求人：[原被申请人]
被反请求人：[原申请人]
反请求事项：[具体请求]

申请书应包含：
1. 当事人信息
2. 反请求事项
3. 事实与理由
4. 与本请求的关系
5. 证据清单
```

## 📋 阶段三：仲裁程序

### 3.1 仲裁员选定策略
```
请制定仲裁员选定策略：

争议类型：[类型]
争议焦点：[焦点]
当事人情况：[背景]

策略分析：
1. 独任仲裁员vs三人仲裁庭
2. 仲裁员专业背景要求
3. 边裁选定策略
4. 首席仲裁员考量
5. 回避情形预判
6. 选定名单建议
```

### 3.2 仲裁庭组成异议
```
请起草仲裁庭组成异议书：

异议事项：[回避申请/其他]
异议理由：[具体理由]

异议书应包含：
1. 异议人信息
2. 仲裁案件信息
3. 异议事项
4. 事实与理由
5. 证据支持
6. 异议结论
```

### 3.3 管辖权异议
```
请起草管辖权异议书：

异议理由：[仲裁协议无效/超出范围/等]

异议书应包含：
1. 异议人信息
2. 案件信息
3. 异议请求
4. 事实与理由
   - 仲裁协议无效情形
   - 超出仲裁范围
   - 仲裁机构无权管辖
5. 法律依据
6. 异议结论
```

## 📋 阶段四：仲裁庭审

### 4.1 仲裁庭审提纲
```
请制作仲裁庭审提纲：

争议焦点：[焦点1/焦点2/焦点3]
我方立场：[立场]

提纲内容：
1. 开场陈述要点
2. 举证顺序
3. 质证策略
4. 辩论要点
5. 最后陈述
```

### 4.2 仲裁代理词
```
请撰写仲裁代理词：

代理方：[申请人/被申请人]
代理思路：[思路]

代理词结构：
1. 首部
2. 案件事实陈述
3. 争议焦点分析
4. 法律适用论证
5. 仲裁请求/答辩支持
6. 结论
```

### 4.3 仲裁调解方案
```
请制定仲裁调解方案：

案件情况：[简述]
当事人意愿：[是否愿意调解]

方案内容：
1. 调解可行性分析
2. 调解底线确定
3. 让步空间评估
4. 调解策略
5. 调解书条款建议
6. 调解不成预案
```

## 📋 阶段五：仲裁裁决

### 5.1 裁决书分析
```
请分析以下仲裁裁决书：

[粘贴裁决书]

分析维度：
1. 裁决结果评价
2. 事实认定分析
3. 法律适用评价
4. 仲裁程序评价
5. 超裁/漏裁审查
6. 社会公共利益审查
7. 救济途径建议
```

### 5.2 裁决撤销申请
```
请起草撤销仲裁裁决申请书：

申请人：[当事人]
仲裁机构：[机构]
裁决日期：[日期]

申请书应包含：
1. 申请人信息
2. 仲裁案件信息
3. 撤销请求
4. 撤销理由（法定情形）
   - 无仲裁协议
   - 超裁/漏裁
   - 程序违法
   - 证据伪造
   - 对方隐瞒证据
   - 仲裁员不当行为
   - 违背社会公共利益
5. 证据清单
6. 法律依据
```

### 5.3 不予执行抗辩
```
请起草不予执行仲裁裁决抗辩书：

被申请人：[当事人]
执行申请人：[对方]

抗辩书应包含：
1. 被申请人信息
2. 执行案件信息
3. 不予执行请求
4. 不予执行理由
5. 证据支持
6. 抗辩结论
```
