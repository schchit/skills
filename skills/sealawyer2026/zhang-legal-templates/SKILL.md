# 法律文书模板库 📄

> 100+高质量法律文书模板 - 合同、诉状、协议一键生成

## 🎯 模板分类

### 合同类（50+）
- **买卖合同**：货物买卖、房产买卖、股权买卖
- **租赁合同**：房屋租赁、设备租赁、土地租赁
- **借款合同**：民间借贷、金融借款、担保合同
- **劳动合同**：全日制、非全日制、劳务合同
- **合伙协议**：普通合伙、有限合伙、隐名合伙
- **投资协议**：股权投资、债权投资、对赌协议
- **技术合同**：技术开发、技术转让、技术服务
- **建设工程**：施工合同、设计合同、监理合同

### 诉讼类（30+）
- **起诉状**：民事起诉状、行政起诉状、刑事自诉状
- **答辩状**：民事答辩状、行政答辩状
- **上诉状**：民事上诉状、行政上诉状
- **申请书**：财产保全、证据保全、强制执行
- **代理词**：一审代理词、二审代理词、再审代理词
- **辩护词**：刑事辩护词、量刑辩护词

### 协议类（20+）
- **离婚协议**：财产分割、子女抚养、债务处理
- **继承协议**：遗产分割、遗嘱执行、放弃继承
- **和解协议**：民事和解、执行和解、劳动和解
- **保密协议**：NDA、竞业限制、商业秘密
- **授权委托**：一般授权、特别授权、转委托

## 📋 核心功能

### 1.1 智能合同生成
```
请根据以下信息生成合同：

【合同类型】：[买卖合同/租赁合同/借款合同等]

【当事人信息】

地址：[ ]
法定代表人/身份证号：[ ]

乙方（受让方）：[名称/姓名]
地址：[ ]
法定代表人/身份证号：[ ]

【交易标的】
标的名称：[ ]
规格型号：[ ]
数量：[ ]
质量标准：[ ]

【交易条件】
价款总额：[ ]元
支付方式：[一次性/分期]
交付时间：[ ]
交付地点：[ ]

【特殊要求】
- 是否需要担保：[ ]
- 是否有违约金条款：[ ]
- 争议解决方式：[诉讼/仲裁]
- 其他特殊约定：[ ]

请生成：
1. 完整合同文本（符合《民法典》规定）
2. 风险提示（关键条款说明）
3. 签署注意事项
4. 配套文件清单（如需担保、授权等）
```

### 1.3 协议条款优化
```
请对以下合同条款进行优化：

【原条款】
[粘贴需要优化的条款]

【当事人地位】
- 我方是：[甲方/乙方]
- 谈判地位：[强势/均衡/弱势]

【优化目标】
- 主要关注点：[风险控制/利益最大化/平衡双方]
- 特殊要求：[ ]

请提供：
1. 优化后的条款（3个版本：保守版、平衡版、进取版）
2. 条款修改说明（修改理由）
3. 风险提示（各版本利弊）
4. 谈判建议（如何争取有利条款）
```
