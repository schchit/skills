# Stock Research Framework

一套系统化的股票投研分析框架，覆盖从基本面到投资决策的 9 个核心维度。

## 🎯 Overview

这是一个 Claude Code Skill，提供了一套全面的股票投资研究模板。当需要分析任何股票时，它会自动引导完成 9 个维度的深度分析，确保分析全面、逻辑清晰、决策有据。

## 📊 9点分析框架

```
┌─────────────────────────────────────────────────────────────┐
│                    投研分析框架                               │
├─────────────────────────────────────────────────────────────┤
│  1. 公司基本面    2. 财务健康度    3. 竞品对比              │
│  4. 股价走势      5. 宏观环境      6. 市场情绪              │
│  7. 最新财报      8. 成长与风险    9. 投资建议              │
└─────────────────────────────────────────────────────────────┘
```

## 🚀 安装

### 从 Claude Code Plugin Marketplace 安装

```bash
# 1. 添加 marketplace
claude plugin marketplace add alondai/claude-plugin-marketplace

# 2. 安装插件
claude plugin install stock-research-framework@alondai
```

### 手动安装

将 `skills/stock-research-framework/SKILL.md` 复制到你的 Claude Code skills 目录中。

## 💡 使用方法

安装后，直接询问即可：

```
帮我分析一下 NVDA
深度研究一下腾讯的投资价值
分析 TSLA 并给出投资建议
```

框架会自动识别股票分析需求，并引导完成 9 个维度的全面分析。

## 🔍 分析维度

| 维度 | 核心内容 |
|------|---------|
| **1. 公司基本面** | 业务架构、市占率、盈利构成、竞争壁垒 |
| **2. 财务健康度** | 营收、净利润、毛利率、资产负债、现金流 |
| **3. 竞品对比** | 3年营收增速、市占率、盈利水平、优劣势 |
| **4. 股价走势** | 历史走势、牛熊表现、波动率、驱动逻辑 |
| **5. 宏观环境** | 货币政策、监管、地缘政治、经贸环境 |
| **6. 市场情绪** | 新闻、研报、舆情、多空倾向 |
| **7. 最新财报** | 营收、利润、现金流、预期偏离度 |
| **8. 成长与风险** | 1-3年增长驱动、潜在风险、估值逻辑 |
| **9. 投资建议** | 估值区间、分类型建议、核心逻辑 |

## 📝 输出格式

分析完成后会输出包含以下内容的完整报告：

- 📊 **数据表格对比** - 清晰的财务指标对比
- 🟢🔴 **正负面影响标注** - 一目了然的利弊分析
- ⚠️ **风险提示** - 全面的风险评估
- 💡 **具体可操作建议** - 明确的买入/持有/观望建议
