# 股票深度投研分析框架

## Overview

一套系统化的股票投研分析框架，覆盖从基本面到投资决策的9个核心维度。适用于任何股票的深度投研分析，确保分析全面、逻辑清晰、决策有据。

## When to Use

- 分析任何股票投资标的
- 需要全面的投研报告
- 做投资决策前的尽职调查
- 对比多个投资标的
- 定期复盘持仓股票

## 1. 公司整体基本面拆解

### 分析目标
> 帮我深度拆解A公司的主营业务架构、行业市场占有率与盈利构成，同时梳理其核心竞争壁垒与差异化优势

### 分析要点
| 维度 | 核心问题 | 关键指标 |
|-----|---------|---------|
| **主营业务架构** | 公司靠什么赚钱？各业务占比如何？ | 收入构成、各业务板块占比 |
| **行业市场占有率** | 在行业中处于什么位置？ | 市占率、行业排名 |
| **盈利构成** | 利润来自哪里？ | 毛利率、各业务盈利贡献 |
| **核心竞争壁垒** | 护城河是什么？ | 品牌、技术、网络效应、牌照、专利 |
| **差异化优势** | 与竞争对手的核心差异？ | 独特定位、独家资源、先发优势 |

### 输出模板
```
### 主营业务架构
| 业务板块 | 核心内容 | 收入占比 | 盈利贡献 |
|---------|---------|---------|---------|
| 业务A | 描述 | XX% | XX% |

### 行业市场占有率
- 市场份额：XX%（行业第X）
- 行业规模：XX亿
- 主要竞争对手：XXX

### 核心竞争壁垒与差异化优势
1. 壁垒A：具体描述
2. 壁垒B：具体描述
3. 差异化优势：XXX
```

## 2. 财务健康度评估

### 分析目标
> 帮我复盘A公司近连续季度的核心财务指标，涵盖营收、归母净利润、毛利率及资产负债结构，简要分析数据变动趋势与财务健康度

### 核心指标清单
| 类别 | 指标 | 健康标准 | 关注点 |
|-----|------|---------|-------|
| **营收** | 营业收入 | 持续增长 | 增速趋势 |
| **归母净利润** | 归属于母公司股东的净利润 | 正值且增长 | 盈利质量 |
| **毛利率** | 毛利率 | 稳定或提升 | 成本控制能力 |
| **资产负债结构** | 资产负债率、流动比率、速动比率 | 负债<60%，流动>1 | 偿债能力 |
| **现金流** | 经营性现金流 | 正值 | 现金造血能力 |

## 5. 宏观&行业环境影响研判

### 分析目标
> 帮我分析当前宏观与行业环境中，对A公司业绩形成核心影响的因子，涵盖货币政策、行业监管、地缘政治及国际经贸环境，解读各因子的正向/负向传导影响

### 影响因子框架
| 因子类别 | 具体因子 | 分析要点 | 传导机制 |
|---------|---------|---------|---------|
| **货币政策** | 利率、流动性、QE/QT | 对估值和融资成本的影响 | 利率→折现率→估值 |
| **行业监管** | 政策法规、牌照、合规要求 | 对业务模式和准入门槛的影响 | 监管→竞争格局→盈利 |
| **地缘政治** | 制裁、贸易限制、政治风险 | 对供应链和市场的影响 | 地缘→成本/市场→业绩 |
| **国际经贸环境** | 关税、汇率、全球需求 | 对出口/进口业务的影响 | 经贸→成本/需求→营收 |

### 输出模板
```

| 因子 | 当前状态 | 影响方向 | 传导机制 |
|-----|---------|---------|---------|
```

## Common Mistakes

| 错误 | 正确做法 |
|-----|---------|
| 只看利好忽视风险 | 每个分析都要有正反两面 |
| 财报只看营收 | 必须看归母净利润、经营性现金流 |
| 忽略宏观环境 | 货币政策/监管/地缘政治对估值影响巨大 |
| 盲目跟随研报 | 分析师也有利益冲突，需独立判断 |
| 估值只看PE | 结合PS、PB、DCF多维度 |
| 没有偏离度分析 | 财报必须对比市场一致预期 |
| 没有给出明确建议 | 必须给出可操作的投资建议 |
| 忽略时间维度 | 增长驱动要区分短期/中期/长期 |

## 输出格式规范

1. **使用表格**：对比数据、财务指标等用表格呈现
2. **使用emoji**：🟢正面 🔴负面 🟡中性 ⚠️风险 ✅亮点
3. **结构清晰**：每个大点用分隔线区分
4. **数据驱动**：尽量用具体数字而非模糊描述
5. **偏离度量化**：财报分析必须量化与预期的偏差
6. **风险提示**：报告末尾必须有风险提示
