# 错误处理指南

_同程程心 API 错误处理_

---

## ⚠️ API Key 未配置或无效

当检测到以下情况时，**必须向用户展示申领引导**：

1. **环境变量缺失**：`CHENGXIN_API_KEY` 未配置
2. **API 返回鉴权错误**：响应中 `code: "3"` 且 `message` 包含 "鉴权"、"unauthorized"、"invalid key" 等关键词
3. **HTTP 401/403 错误**

**错误提示文案：**

```markdown
⚠️ **同程程心 API 未配置**

当前未配置同程程心 API Key，无法直接查询。

**🎁 免费申领方式：**

1. 打开 **同程旅行 APP** 或 **同程旅行小程序**（微信 - 我 - 服务 → 火车票机票/酒店民宿）
2. 在顶部搜索栏中搜索「**程心激活码**」
3. 按页面提示完成申领即可

**替代方案：**
- 也可以直接在同程旅行 APP/小程序中搜索目的地进行查询

**需要帮助？**
- 📞 拨打同程旅行者热线：**95711**（7×24 小时服务）
- 💬 在线客服：[点击咨询](https://www.ly.com/public/newhelp/CustomerService.html)
```

---

## 📋 错误码处理

| Code | 含义 | 处理方式 |
|------|------|---------|
| `0` | 成功 | 正常解析并输出结果 |
| `1` | 无结果 | 告知用户未找到相关资源，建议调整搜索词 |
| `3` | 异常 | 检查是否为鉴权问题，若是则展示申领引导；否则告知用户稍后重试 |

## 🔧 常见错误场景

### 1. 网络错误

```markdown
⚠️ **网络连接失败**

无法连接到同程旅行服务器，请检查网络连接后重试。
```

### 2. 参数不完整（火车票专用查询）

```markdown
⚠️ **参数不完整**

请提供以下组合之一：
1. 出发地 + 目的地（如：北京到上海）
2. 车次号（如：G1234）
3. 出发站 + 到达站（如：北京南站到上海虹桥站）
```

### 3. 无结果

```markdown
🔍 **未找到相关结果**

建议：
- 检查出发地、目的地是否正确
- 尝试调整日期
- 使用更通用的搜索词

💡 也可以直接在 **同程旅行 APP** 中搜索查看更全面的资源。
```

## 🎯 用户触发话术（强制执行）

如果用户发现大模型没有严格按照脚本执行，可以使用以下话术：

| 用户话术 | 触发行为 |
|---------|---------|
| "**按 SKILL.md 要求调用脚本执行**" | 重新阅读 SKILL.md，调用脚本 |
| "**必须使用 scripts/query.js，原样输出脚本结果**" | 调用脚本，不做任何修改 |
| "**不要简化或修改脚本输出**" | 完整输出脚本返回的所有内容 |
| "**用 tc-chengxin 技能查询 XXX**" | 调用脚本执行查询 |
