# 股市小灵通-信息查询基础版本 Skill

这是股市小灵通-信息查询基础版本 Skill 的发布目录，仅包含 Skill 文档与发布元信息，不包含后端服务代码。

## 能力范围

- A 股/港股行情查询（日/周/月、指标、资金流向、交易日历）
- 基本面查询（公司资料、财报、财务指标、分红）
- 指数查询（行情、权重、成分股、分类）
- 文本数据查询（董秘问答、政策法规、研报元数据）

## 依赖

- 可访问的股票查询 API 服务（`base_url` 见 `SKILL.MD`）
- 有效的 `TAX-API-Key`（通过平台密钥配置注入）

## TAX-API-Key 免费获取

1. 使用电脑端浏览器访问 `https://tax.yyyou.top/`。
2. 微信扫码后自动注册/登录。
3. 免费创建授权码并复制。
4. 将授权码配置到平台的技能密钥/环境变量（建议键名：`TAX_API_KEY`），不要在聊天消息里明文发送。

## 安装方式

通过 Clawhub 技能市场页面安装本 Skill（不要求执行第三方命令下载）。

## 安全边界

- 只读查询，不调用写操作
- 不访问用户本地文件或系统
- 不保存用户隐私数据

## 发布前检查

- `SKILL.MD` 中接口路径、鉴权方式与线上服务保持一致
- `base_url` 为线上可访问域名
- 示例请求可在目标环境实际跑通
- 文档中不包含密钥、账号、内网地址等敏感信息
- 已在平台技能配置中声明并注入 `TAX_API_KEY`（不通过聊天传递）
- `TAX_API_KEY` 已限定为最小权限（只读查询）、可撤销，并启用周期性轮换策略
