# Epoch
Epoch time — fast. Convert, calculate, compare Unix timestamps.
## Commands
- `now` — Current epoch time
- `date <epoch>` — Epoch to human date
- `epoch <date_string>` — Date to epoch
- `ago <duration>` — What time was N ago
- `zones` — Current time in world timezones
- `calc <epoch> +|- <seconds>` — Epoch arithmetic
## Usage Examples
```bash
epoch now
epoch date 1700000000
epoch epoch "2024-01-01"
epoch ago "2h 30m"
epoch zones
```
