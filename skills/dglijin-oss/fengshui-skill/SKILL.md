### 四、命卦计算

**公式（简化）：**
- 男命：(100 - 出生年份后两位) ÷ 9，取余数
- 女命：(出生年份后两位 - 4) ÷ 9，取余数

| 余数 | 命卦 |
|------|------|
| 1 | 坎 |
| 2 | 坤 |
| 3 | 震 |
| 4 | 巽 |
| 5 | 男坤女艮 |
| 6 | 乾 |
| 7 | 兑 |
| 8 | 艮 |
| 9 | 离 |

## 使用方法

### 八宅排盘

```
问：1990 年男，住坐北朝南房子，风水如何？

回复：
【八宅风水分析】

命主：1990 年男
命卦：坎卦（东四命）

宅型：坐北朝南 = 坎宅（东四宅）
宅命匹配：✅ 吉（东四命住东四宅）

【游年九星】
生气（大吉）：东南方 - 利财运事业
天医（吉）：东方 - 利健康贵人
延年（吉）：南方 - 利婚姻事业
伏位（小吉）：北方 - 利平稳守成
祸害（凶）：西方 - 防口舌是非
六煞（凶）：西北方 - 防桃花破财
五鬼（大凶）：东北方 - 防火灾盗贼
绝命（大凶）：西南方 - 防破财疾病

【布局建议】
✅ 卧室：宜在生气、天医、延年方
✅ 大门：宜开在生气、延年方
✅ 厨房：宜压在凶方，灶口向吉方
❌ 卫生间：不宜在吉方

【综合评分】75 分（吉）
```
