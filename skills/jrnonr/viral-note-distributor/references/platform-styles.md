# 各平台图文风格指南

## 小红书

**风格特点：**
- 亲切、口语化、"姐妹们"等人称
- emoji 大量使用（标题/开头/结尾/分段都要有）
- 引流话术："姐妹们"、"真的"、"绝了"、"我天"等
- 共鸣感 + 实用干货结合
- 结尾引导互动（评论区见/你们觉得呢/有同感举手）

**输出规格：**
- 标题：20字内，带 emoji，疑问句或感叹句
- 正文：800-1000字，分段（每段不超过50字）
- 标签：5-8个，#开头

**正文结构：**
1. 开头共鸣句（痛点/爽点切入，带emoji）
2. 核心内容（干货 + 真实体验，每段带emoji点缀）
3. 结尾 CTA（引导评论/收藏/关注）

**引流话术库：**
```
开头引导：
- "姐妹们，今天必须来说说..."
- "真的忍不住了，一定要分享给你们..."
- "我天，这个东西让我..."
- "救命！终于有人说了..."

中段点缀：
- "说实话..." "真的服了..." "绝了..."
- "姐妹们懂的都懂..." "你品，你细品..."
- "重点来了！" "关键是这个..." "但是！"

结尾互动：
- "你们有没有同感？评论区告诉我！"
- "有同感的点个赞🙋"
- "收藏起来慢慢看！" "给我冲！"
- "你们站哪边？评论区说说！"

争议制造：
- "有人说XXX好，我偏要说XXX坏"
- "大家都踩的坑，我来说说真相"
- "真的不是我说，但..."
```

**emoji使用频率：** 每50字至少1个emoji，标题必须带emoji

## 抖音（图文中长图文）

**风格特点：**
- 短平快、信息密度高
- 前3行必须抓眼球（抖音feed只展示前3行）
- 悬念式/数字式标题多
- 情绪饱满，可适度夸张

**输出规格：**
- 标题：15字内，制造悬念或好奇心
- 正文：300-500字，简洁有力
- 标签：3-5个热门话题标签

**正文结构：**
1. 钩子开头（震惊/好奇/共鸣）
2. 核心信息（1-2个关键点）
3. 引导互动（你们遇到过吗/评论区告诉我）

## B站

**风格特点：**
- 真实、个人风格强
- 可以稍微长一点，有代入感
- 半干货半分享感
- 语言可以稍微活泼/鬼畜，但不要太过

**输出规格：**
- 标题：25字内，可带括号补充（【XX】）
- 正文：1000-1500字，可分段小标题
- 标签：5-8个，站内热门话题优先

**正文结构：**
1. 个人故事切入（为什么做/怎么发现的）
2. 核心分享（真实体验 + 细节）
3. 总结 + 推荐理由

---

## 公众号

**风格特点：**
- 深度、干货、逻辑严密
- 标题严肃但可带数字（如"3个技巧"）
- 正文可以有段落标题
- 结尾引导关注/点在看

**输出规格：**
- 标题：20-30字，可含数字
- 正文：1500-2000字，分段落
- 标签：1-2个主标签（公众号标签机制不同）

**正文结构：**
1. 热点/痛点引入（有数据/有共鸣）
2. 核心观点（分1/2/3点展开）
3. 总结 + 引导点在看 + 关注
