# 大小盘风格轮动分析 Skill

分析A股市场大小盘风格轮动情况，通过波动差值判断市场风格偏向和切换信号。

## Overview（功能概述）

使用大虾皮 API 获取中证2000（小盘）与沪深300（大盘）的波动差值数据，分析风格轮动趋势，判断当前市场风格偏向。

## When to Use（何时使用）

当用户需要以下功能时触发：

- 判断当前市场风格偏向（大盘股 vs 小盘股）
- 分析大小盘风格轮动趋势
- 识别风格切换信号
- 了解市场资金流向（游资 vs 机构）

**具体触发词**：大小盘风格、风格轮动、大盘股小盘股、风格切换、沪深300、中证2000、大盘小盘

## When Not to Use（何时不使用）

以下场景不应使用本 Skill：

- 个股层面的买卖点、基本面或财报解读
- 行业板块层面的分析
- 债券、基金、期货、外汇等非风格分析
- 需要实时盘中信号或自动交易指令的请求

## Process（流程主体）

### Step 0: 前期准备

**触发条件**：首次使用或 Token 未配置

**跳过条件**：Token 已配置且有效，确认已经配置了Token则跳过该步骤

**完成标准**：API 调用正常，无认证错误

**执行步骤**：

**步骤 0.1：检查 Token 配置状态**

```bash
npx daxiapi-cli@latest config get token
```

**步骤 0.2：如未配置，获取 Token**

1. 提示用户访问 [daxiapi.com](https://daxiapi.com) 个人主页
2. 开通 API Token 功能
3. 获取生成的 Token

**步骤 0.3：配置 Token**

```bash
# 方式一：通过 CLI 配置（推荐）
npx daxiapi-cli@latest config set token YOUR_TOKEN_FROM_DAXIAPI

# 方式二：设置环境变量
export DAXIAPI_TOKEN=YOUR_TOKEN_FROM_DAXIAPI
```

**步骤 0.4：验证配置**

```bash
npx daxiapi-cli@latest market style
```

如返回正常数据，则配置成功。

### Step 1: Agent 循环（获取+分析）

**目标**：获取大小盘风格轮动数据并进行深入分析

**推荐维度**（根据用户需求选择）：

| 维度         | CLI 命令                                   | 分析重点               | 优先级 |
| ------------ | ------------------------------------------ | ---------------------- | ------ |
| 波动差值     | `npx daxiapi-cli@latest market style`      | 大小盘相对强弱         | 高     |
| 趋势变化     | 分析最近 20 日差值变化            | 风格切换信号           | 高     |
| 均值回归     | 判断差值是否接近极端值            | 风格反转可能性         | 中     |

**核心指标说明**：

**大小盘波动差值** = 中证2000涨跌幅 - 沪深300涨跌幅

| 差值情况      | 含义                                   |
| ------------- | -------------------------------------- |
| 差值 > 0      | 小盘股强于大盘股，游资更积极           |
| 差值 < 0      | 大盘股强于小盘股，机构更积极           |
| 差值 ≈ ±10%   | 可能出现均值回归，风格切换信号         |

**循环流程**：

**每次循环包含**：

1. **获取数据**
    - 执行 CLI 命令：`npx daxiapi-cli@latest market style`
    - 检查数据完整性（日期、差值、趋势）
    - 处理异常情况

2. **分析数据**
    - 判断当前风格偏向
    - 分析最近 20 日差值变化趋势
    - 识别风格切换信号
    - 评估均值回归可能性

**灵活性**：

- ✅ 可根据数据质量调整分析深度
- ✅ 可结合市场环境解读风格含义
- ✅ 可根据用户需求调整建议重点

**完成标准**：

- 风格数据已获取
- 风格偏向已判断
- 趋势分析已完成
- 关键发现已记录

## Common Pitfalls（常见陷阱）

| 陷阱           | 说明                       | 避免方法                   |
| -------------- | -------------------------- | -------------------------- |
| 绝对化表述     | 使用"必然切换"等表述       | 改为"可能"、"大概率"       |
| 单一时点判断   | 只看当前差值不看趋势       | 结合 20 日趋势分析         |
| 忽视市场环境   | 只看差值不看宏观环境       | 提醒关注宏观因素          |
| 过度预测       | 对风格切换做确定性预测     | 改为概率判断              |
| 均值回归误区   | 认为极端值一定会回归       | 说明可能继续分化          |

## Gotchas（避坑案例）

- **401 认证失败（Token 缺失或失效）**
  - 错误表现：`401 Unauthorized` 或 `Authentication failed`
  - 修复动作：执行 `npx daxiapi-cli@latest config get token` 检查配置；若为空或失效，执行 `npx daxiapi-cli@latest config set token YOUR_TOKEN_FROM_DAXIAPI` 重新配置

- **空数据返回（非交易日/更新延迟）**
  - 错误表现：返回空数据或数据日期过旧
  - 修复动作：向用户说明数据时效限制，建议在收盘后再查询

- **429 请求频率超限**
  - 错误表现：`429 Too Many Requests`
  - 修复动作：等待 30-60 秒后重试

## Key Principles（重要原则）

1. **结论先行**：开头即给出风格偏向判断
2. **趋势结合**：不只看当前差值，结合趋势分析
3. **中性表述**：不使用绝对化词汇
4. **数据支撑**：所有结论引用具体数据
5. **免责声明**：所有报告必须包含免责声明
