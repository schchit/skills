# 牛津大学 (University of Oxford)

英语世界最古老的大学，世界顶尖研究型大学。

## 学校介绍

**牛津大学**成立于 1096 年，位于英国牛津市，是英语世界最古老的大学。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1096 年 |
| **地点** | 英国牛津 |
| **类型** | 公立研究型大学 |
| **校训** | Dominus illuminatio mea (主是我的光) |
| **学院数** | 39 个 |
| **排名** | 世界前 5 |

### 知名学院

- **基督堂学院** - 最大学院
- **贝利奥尔学院** - 最古老学院
- **莫德林学院** - 最美学院
- **万灵学院** - 仅研究生

### 申请要求

- A-Level: A*A*A - AAA
- IB: 38-40 分
- 雅思：7.5 分 (单项 7.0)
- 入学笔试 + 面试
