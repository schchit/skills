# New Balance 新百伦技能

## 技能描述
New Balance（新百伦）是美国知名运动品牌，以跑鞋、运动鞋闻名，提供多种宽度选择。本技能帮助用户查询 New Balance 产品、门店、运动装备信息。

## 核心功能
- 👟 **产品查询**：跑鞋、运动鞋、服装、配件
- 🏪 **门店查询**：全球零售店位置
- 📅 **新品发布**：最新系列资讯
- 🏃 **运动类别**：跑步、篮球、网球、健身等
- 💰 **价格查询**：产品价格范围
- 🌟 **经典系列**：574、990、996 等

## 使用场景
- 查询 New Balance 最新产品
- 了解跑鞋科技
- 查找附近门店
- 获取鞋码建议（多种宽度）
- 对比其他运动品牌

## 注意事项
- New Balance 是美国运动品牌
- 成立于 1906 年，总部位于波士顿
- 以提供多种鞋宽著称
