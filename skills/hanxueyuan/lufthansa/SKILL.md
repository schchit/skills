# 汉莎航空 (Lufthansa)

德国最大、欧洲领先的航空公司。

## 公司介绍

**汉莎航空公司**成立于 1953 年，总部位于德国科隆，是欧洲最大的航空集团。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1953 年 1 月 6 日 |
| **总部** | 德国科隆 |
| **枢纽机场** | 法兰克福、慕尼黑 |
| **机队数量** | 300+ 架 |
| **目的地** | 200+ 城市 |
| **联盟** | 星空联盟 (创始成员) |

### 舱位等级

| 舱位 | 介绍 |
|------|------|
| **头等舱** | 顶级奢华体验 |
| **商务舱** | 高端商务出行 |
| **优选经济舱** | 舒适经济舱 |
| **经济舱** | 标准经济舱 |

### 会员计划 - 汉莎常旅客 (Miles & More)

| 等级 | 要求 | 权益 |
|------|------|------|
| **会员** | 注册即享 | 里程累积 |
| **Frequent Traveller** | 3.5 万英里 | 优先值机、额外行李 |

### 里程权益

- **里程累积**: 飞行、信用卡、合作伙伴
- **里程兑换**: 免费机票、升舱、酒店
- **家庭账户**: 里程共享
- **里程购买**: 直接购买里程
- **里程捐赠**: 慈善捐赠

### 中国航线

- **通航城市**: 北京、上海、广州、香港
- **航班频率**: 每日多班
- **机型**: A380、B747-8、A350 等

### 特色服务

- **汉莎贵宾室**: 全球多地贵宾室
- **机上餐饮**: 德国美食体验
- **机上娱乐**: 丰富娱乐系统
- **宠物运输**: 专业宠物服务
- **特殊餐食**: 多种特殊餐食选择
