# 凯悦酒店 (Hyatt Hotels)

美国知名高端酒店集团。

## 集团介绍

**凯悦酒店集团**成立于 1957 年，总部位于美国芝加哥，是全球领先的高端酒店管理公司。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1957 年 |
| **总部** | 美国芝加哥 |
| **酒店数量** | 1200+ 家 |
| **覆盖国家** | 70+ |
| **客房数量** | 25 万 + |

### 旗下品牌

| 品牌 | 定位 |
|------|------|
| **柏悦 (Park Hyatt)** | 顶级奢华 |
| **宝格丽 (Bvlgari)** | 超奢华 (合作) |
| **君悦 (Grand Hyatt)** | 豪华酒店 |
| **凯悦 (Hyatt Regency)** | 高端商务 |
| **安达仕 (Andaz)** | 时尚设计 |
| **凯悦嘉轩 (Hyatt Place)** | 中端精选 |
| **凯悦嘉寓 (Hyatt House)** | 长住公寓 |
| **Thompson Hotels** | 精品酒店 |
| **Destination Hotels** | 度假村 |

### 会员计划 - 凯悦天地 (World of Hyatt)

| 等级 | 要求 | 权益 |
|------|------|------|
| **会员** | 注册即享 | 积分累积、免费 WiFi |
| **探索者** | 5 晚/年 | 延迟退房、优先入住 |
| **冒险家** | 10 晚/年 | 房间升级、欢迎礼品 |
| **环球客** | 30 晚/年 | 行政酒廊、套房升级 |
| **环球客探索者** | 60 晚/年 | 行政套房、专属礼遇 |

### 积分权益

- **积分累积**: 住宿、餐饮、会议
- **积分兑换**: 免费住宿、航空里程
- **积分转赠**: 会员间转移
- **积分购买**: 直接购买积分
- **积分 + 现金**: 混合支付选项

### 中国布局

- **酒店数量**: 100+ 家
- **覆盖城市**: 北京、上海、广州、深圳、成都等
- **主要品牌**: 柏悦、君悦、凯悦、安达仕

### 特色服务

- **凯悦餐饮**: 知名餐厅和酒吧
- **会议设施**: 商务会议服务
