# 哥伦比亚大学 Columbia 技能

## 技能描述
哥伦比亚大学（Columbia University）是美国纽约市的私立研究型大学，常春藤盟校成员，世界顶尖高等学府，位于纽约曼哈顿。本技能帮助用户查询哥伦比亚大学相关信息。

## 核心功能
- 🎓 **学校信息**：历史沿革、校园介绍、排名情况
- 📚 **学院设置**：哥伦比亚学院、工程学院、商学院、法学院、新闻学院等
- 📝 **招生信息**：本科申请、研究生申请、录取要求
- 💰 **学费资助**：学费标准、奖学金、助学金
- 🏛️ **知名校友**：政界（多位美国总统）、商界、媒体界杰出校友
- 📊 **学术实力**：新闻、商科、法律、国际关系等优势学科

## 使用场景
- 查询哥伦比亚大学基本信息
- 了解申请要求和流程
- 查询学费和奖学金信息
