# Seagate 希捷技能

## 技能描述
Seagate（希捷）是全球领先硬盘制造商，以机械硬盘、固态硬盘、NAS 存储闻名。本技能帮助用户了解希捷产品、技术、选购建议。

## 核心功能
- 💾 **硬盘查询**：机械硬盘、固态硬盘
- 🏠 **家用存储**：Backup Plus、Expansion
- 🖥️ **企业存储**：Exos、IronWolf
- 💰 **价格查询**：产品定价、对比
- 🔧 **技术支持**：保修、数据恢复
- 📊 **技术规格**：容量、速度、接口

## 使用场景
- 查询希捷产品信息
- 获取购买建议
- 了解存储技术
- 技术支持帮助
