# Paint

Digital paint toolkit — color palette generation, gradient creation, texture generation, pattern making, and color space conversion.

## Commands

| Command | Description |
|---------|-------------|
| `paint run` | Execute main function |
| `paint list` | List all items |
| `paint add <item>` | Add new item |
| `paint status` | Show current status |
| `paint export <format>` | Export data |
| `paint help` | Show help |

## Usage

```bash
# Show help
paint help

# Quick start
paint run
```

## Examples

```bash
# Run with defaults
paint run

# Check status
paint status

# Export results
paint export json
```

- Run `paint help` for all commands
paint/`

## When to Use

- for batch processing paint operations
- as part of a larger automation pipeline

## Output

Returns results to stdout. Redirect to a file with `paint run > output.txt`.
