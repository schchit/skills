# Journal Prompt

日记引导助手。每日写作引导、感恩日记、反思日记、晨间日记、晚间总结、周总结模板。Journal prompts for daily writing, gratitude, reflection, morning pages, evening review, weekly summary. 日记、写作、反思、感恩。

## 常见问题

**Q: 这个工具适合谁用？**
A: 任何需要journal prompt的人，无论是个人还是企业用户。

**Q: 输出格式是什么？**
A: 主要输出Markdown格式，方便复制和编辑。

## 可用命令

- **prompt** — prompt
- **gratitude** — gratitude
- **reflect** — reflect
- **morning** — morning
- **evening** — evening
- **weekly** — weekly

## 专业建议

- 不追求完美，写就对了
- 固定时间写(晨间或睡前)
- 分钟就够，质量>长度
- 诚实面对自己的感受
- 晨间: 意图+3个重点+感恩
