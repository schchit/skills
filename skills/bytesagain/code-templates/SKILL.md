# Claude Code Templates

CLI tool for configuring and monitoring Claude Code ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from davila7/code-templates

## Usage

Run any command: `code-templates <command> [args]`
