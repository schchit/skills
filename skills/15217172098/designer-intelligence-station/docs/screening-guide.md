## 📊 核心筛选逻辑

基于对用户实际筛选行为（120+ 条数据）的分析，筛选逻辑遵循**三层判断**：

> **「设计相关性 × 实用性 × 创新性」三维评分**

| 维度 | 评分标准 | 权重 |
|------|---------|------|
| **设计相关性** | 是否直接影响设计工作？ | 40% |
| **实用性** | 能否直接用/参考？ | 35% |
| **创新性** | 是否有新交互/新洞察？ | 25% |

**综合得分≥70%** → 采用  
**综合得分<70%** → 排除

## 🔧 6 维筛选标准

| 维度 | 权重 | 判断标准 | 采用率 | 典型示例 |
|------|------|---------|--------|---------|
| **[1] 交互/视觉创新** | ⭐⭐⭐ | 专利（有创新）/新交互/视觉技术 | 100% | 苹果手势专利、Apple Pencil Pro 物理模拟、QR tornado |
| **[2] 设计工具/资源** | ⭐⭐⭐ | 提升设计师效率的工具/资源 | 85% | Aesthetron AI、Mobbin、AI Interaction Atlas |
| **[3] 深度分析/洞察** | ⭐⭐⭐ | 趋势/方法论/行业洞察 | 85% | 消亡的动态磁贴、蓝紫渐变色分析、State of Designer |
| **[4] 大厂 AI/设计战略** | ⭐⭐ | 苹果/谷歌/华为的 AI 或设计发布 | 60% | 谷歌 Android AI、CarPlay 开放 AI 控制 |

| **[6] AI+ 设计工作流** | ⭐⭐⭐ | AI 工具/工作流创新 | 80% | Cursor 设计哲学、Anthropic Skills 指南 |

## 🎯 边界情况判断

### 判断流程

```
1. 是否是专利/交互细节？
   ├─ 是 + 交互创新 → ✅ 采用（100%）
   ├─ 是 + 常规专利 → ❌ 排除
   └─ 否 → 继续

2. 是否是设计工具/资源？
   ├─ 是 → ✅ 采用（85%）
   └─ 否 → 继续

3. 是否是深度文章/分析？
   ├─ 是 → ✅ 采用（85%）
   └─ 否 → 继续

4. 是否是 AI+ 设计工作流？
   ├─ 是 → ✅ 采用（80%）
   └─ 否 → 继续

5. 是否是大厂 AI/设计战略？
   ├─ 是 → ✅ 采用（60%）
   └─ 否 → 继续

6. 是否是设计作品/案例？
   ├─ 是 + 创意/趣味 → ✅ 采用（70%）
   ├─ 是 + 常规展示 → ❌ 排除
   └─ 否 → 继续

7. 是否是纯资讯/公告？
   ├─ 是 + 设计影响 → ✅ 采用
   ├─ 是 + 无设计影响 → ❌ 排除
   └─ 否 → 继续

8. 是否是社区八卦/教程/营销？
   ├─ 是 → ❌ 排除
   └─ 否 → 边界判断
```

### 边界案例解析

| 内容 | 判定 | 关键区别 |
|------|------|---------|
| ✅ Figma×Claude Code | 采用 | 设计工具演进，影响工作流 |
| ❌ Stitch 设计系统更新 | 排除 | 常规产品更新，无实质创新 |
| ✅ 微信扫一扫改版 | 采用 | 功能创新，影响用户体验 |
| ❌ Claude Cowork 升级 | 排除 | 企业级功能，与设计关联弱 |
| ✅ 三星防窥屏 | 采用 | 硬件创新，影响交互设计 |
| ✅ OpenAI 硬件全家桶 | 采用 | 大厂战略，新交互范式 |
| ❌ Mike Matas Ferrari | 排除 | 营销性质，无设计细节 |

```
1. 检查是否满足 S 级条件

3. 检查是否满足 B 级条件

```

## 📝 输出格式要求

### 情报卡片格式

```markdown
### [排名]. [公司/事件]

**标题**: [情报标题](链接)
**维度**: [1][3] | **分级**: S
**来源**: 来源名称
**时间**: 发布时间
**摘要**: 1-2 句核心信息

**设计师意义**: 
- 对设计工作的直接影响
- 可能带来的设计机会
- 需要警惕的风险
```

### 趋势洞察格式

```markdown
## 💡 趋势洞察：[趋势名称]

### 👁️ 观察（事实层）
- 客观描述现象
- 关键信号 1、2、3

### 🧠 分析（解读层）
- 为什么现在发生？
- 设计师视角的解读

### 🔮 预测（预判层）
- 短期影响（1-3 个月）
- 中期影响（3-12 个月）
```

## 📚 记忆口诀

> **一收三不收**
> 
> 收：有设计/技术/趋势价值
> 
> 不收：纯营销、常规更新、参数对比

> **分级看维度**
> 
> [1]+[3] 或 [1]+[6] 或 [3]+[6] = S 级头条
> 
> [1] 或 [2] 或 [3] 或 [6] = A 级重点
> 
> [4] 或 [5] = B 级简讯
> 
> 纯资讯/市场/营销 = 排除

> **六维含义**
> 
> [1] 交互创新：专利、新交互、视觉技术
> [2] 设计工具：提升效率的工具/资源
> [3] 深度洞察：趋势、方法论、行业分析
> [4] 大厂战略：苹果/谷歌/华为的 AI/设计发布
> [5] 设计作品：创意参考、趣味性案例
> [6] AI 工作流：AI 工具、工作流创新
