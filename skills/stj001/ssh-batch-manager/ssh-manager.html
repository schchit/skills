🔐 SSH 公钥管理

📝 使用说明

  1. 点击"读取公钥"查看公钥内容
  2. 点击"复制公钥"复制到剪贴板
  3. 点击"下载公钥"保存为文件
  4. 将公钥内容添加到目标服务器的 ~/.ssh/authorized_keys

🖥️ 服务器端操作

# SSH 到目标服务器
ssh user@host

# 创建 .ssh 目录
mkdir -p ~/.ssh

# 追加公钥
echo "公钥内容" >> ~/.ssh/authorized_keys

# 设置权限
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
