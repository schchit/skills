🔐 SSH Public Key Management

📝 Instructions

  1. Click "Read Key" to view public key content
  2. Click "Copy Key" to copy to clipboard
  3. Click "Download Key" to save as file
  4. Add public key content to target server's ~/.ssh/authorized_keys

🖥️ Server-side Operations

# SSH to target server
ssh user@host

# Create .ssh directory
mkdir -p ~/.ssh

# Append public key
echo "public_key_content" >> ~/.ssh/authorized_keys

# Set permissions
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
