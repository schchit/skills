# /novel track - 综合追踪

## 触发方式

```
/novel track [选项]
/novel track --check    # 写作后追踪验证
```

或对话式：
```
更新进度追踪
检查写作状态
```

---

## 功能

1. **追踪更新**：更新章节进度、字数统计
2. **一致性检查**：验证角色状态、关系变化
3. **情节追踪**：确认线索埋设/揭晓进度
4. **质量验证**：字数是否达标、要素是否齐全

---

## 执行方式

### 写作后自动执行（after_each_chapter）

每次 `/novel write` 完成后 AI 自动执行 `--check`（失败阻断写作）

### 手动触发

```
/novel track
```
