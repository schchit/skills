# 干货内容助手 v3.0

## 核心功能

🔍 **搜索具体干货** - 从网页提取实用的技巧、方法、经验
📝 **生成干货清单** - 5-15 条具体可执行的技巧
✨ **创作文案** - 小红书风格，含表情符号和标签
📤 **自动发布** - 生成封面图，发布到小红书

## 快速使用

```bash
# 搜索职场技巧，10 条干货
python ~/.openclaw/skills/local-life-hotspot/run.py -k "职场技巧" -n 10

# 搜索学习方法，15 条干货，自动发布
python ~/.openclaw/skills/local-life-hotspot/run.py -k "学习方法" -n 15 -a
```

## 参数说明

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--keyword` | `-k` | 搜索关键词 | `职场干货` |
| `--top-n` | `-n` | 干货技巧数量 | `10` |
| `--auto-publish` | `-a` | 自动发布 | `false` |
| `--content` | `-c` | 直接提供内容 | `''` |

## 支持的领域

- 职场：职场技巧、面试技巧、沟通技巧
- 学习：学习方法、考试攻略、知识管理
- 科技：科技知识、AI 工具、技术学习
- 生活：生活小妙招、家居技巧、收纳方法
- 美食：烹饪技巧、美食攻略、菜谱分享
- 健身：健身方法、运动技巧、减肥攻略
- 其他：任意领域的技巧/方法/经验

## 内容质量标准

✅ 好的干货：
- 具体可执行
- 有实际指导意义
- 经过实践验证

❌ 避免的内容：
- 空洞的口号
- 模糊的建议
- 热点摘要
